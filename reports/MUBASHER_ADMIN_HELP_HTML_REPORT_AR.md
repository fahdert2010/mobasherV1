# تقرير التشريح الفني والبرمجي المستقل لملف صفحة المساعدة والدعم الساكن المترجم (`help.html`)

**اسم الوثيقة:** MubasherStream Compiled Admin Portal Help & Documentation Static HTML Page Route (.next) Architectural Anatomy & Integration Manual  
**تاريخ التحليل:** ٢٩ يونيو ٢٠٢٦  
**الجهة المعدّة:** خبير الأنظمة والبرمجيات المتقدم للذكاء الاصطناعي (Google AI Studio Coding Agent)  
**حالة التوثيق:** تقرير فني مستقل - معتمد ومحفوظ في مجلد التقارير لـ `MubasherStream/app`

## ١. مقدمة عامة ورؤية هندسية للملف (`help.html`)

يمثل الملف **`MubasherStream/app/.next/server/app/mubasher_admin/help.html`** واجهة العرض الساكنة مسبقة الرندر (Static Pre-rendered HTML Shell) لـ **صفحة مركز المساعدة والدعم الفني والتوثيق المدمج (Admin Help Center & Documentation Static View)** لمنصة البث **مباشر ستريم (MubasherStream)** في بيئة الإنتاج تحت المسار الآمن:  
`/mubasher_admin/help`

يقوم Next.js بتوليد وحفظ هذا الملف كواجهة ساكنة لتسريع وتسهيل وصول مديري البث لأدلة التشغيل ودلائل حلول المشاكل وتكوين خيارات النظام المعقدة دون تحميل خوادم الويب أعباء عمليات المعالجة الديناميكية المتكررة. بعد اكتمال التحميل المادي الفوري للهيكل الساكن، يبدأ محرك React بعملية "الترطيب" (React Hydration)، لتفعيل شريط البحث السريع، وقوائم التبويبات والمستندات التفاعلية، ونماذج الاتصال بقسم المساعدة أو التحقق من النسخ.

---

## ٢. تفكيك وتحليل محتويات ملف العرض الساكن (`help.html`)

يتحلل الهيكل المادي والبرمجي الداخلي لملف `help.html` المترجم إلى المكونات الفنية التالية:

1. **روابط الأصول وروابط التنسيق (Header Markups & CSS Assets):**
   * إقران الواجهة بملفات التنسيق لـ Tailwind CSS لضمان ثبات الهوية الرسومية الساحرة للمنصة والألوان الداكنة (Slate-950 Cosmic Theme).
   * إشارات التحميل المسبق (Preload Indicators) للمحركات والوظائف التفاعلية المسؤولة عن معالجة المدخلات والبحث اللحظي ضمن فهارس الدعم الفني.

2. **الهيكل الرسومي للـ DOM (Document Object Model Structure):**
   * كتل تقسيم المستندات (Grid/Sidebar Layout) التي تعرض القوائم التفصيلية المسبقة الرندر لأقسام المساعدة: إعداد قنوات IPTV، تهيئة معلمات FFmpeg، تشغيل محركات الكاش، وتكوين تكاملات MikroTik.
   * هياكل الأسئلة الشائعة القابلة للتمدد (Accordion FAQ Structures).
   * أزرار تنزيل كتيب التشغيل وتجربة الاتصال بالمخدم.

3. **حقنة فهارس البحث المدمجة وحالة المزامنة (Dehydrated React State Payload):**
   * يحتوي الملف بآخره على كتل `<script id="__NEXT_DATA__" type="application/json">` المحملة بكائنات JSON والتي تحمل فهارس البحث المسبقة لصفحة المساعدة، مما يتيح للمتصفح معالجة استعلامات المشرف عن حلول الأعطال والتحكم محلياً دون الاضطرار لإجراء استعلامات مكثفة أو انتظار الخادم الخلفي.

## ٣. معوقات ملف المساعدة والدعم الساكن في بيئات Docker وسيرفرات QNAP NAS

عند تشغيل مباشر ستريم وعرض واجهة الدعم الساكنة عبر حاويات Docker على سيرفرات **QNAP NAS**، تبرز معوقات تقنية مرتبطة بطبيعة بيئة التشغيل المعزولة:

1. **عقبة تحديث بيانات المساعدة وعرض التعليمات القديمة (Stale Static Information):**
   بما أن الملف ساكن ويتم توليده أثناء مرحلة البناء، فإنه قد يعرض تعليمات، أو روابط اتصال بالخوادم، أو بروتوكولات قديمة لا تواكب حالة النظام الحالية أو التعديلات الشبكية الحقيقية التي أجريت على خادم كيوناب المضيف.
   * **الحل الفني والترحيل السليم:** الاعتماد على تفعيل خدمات الترطيب الديناميكي، وإتاحة قراءة أدلة المساعدة من ملفات Markdown خارجية قابلة للتحديث والتخزين على الأقراص الصلبة الفيزيائية لكيوناب وربطها بالكامل بـ Docker Volume.
2. **عراقيل توجيه الـ Reverse Proxy والمسارات المفقودة للأصول (Asset Path Incompatibility):**
   قد تفشل المتصفحات في تحميل الأوراق التنسيقية والصور ومقاطع الشرح التوضيحية لملف المساعدة إذا تم تكوين الـ Reverse Proxy أو حزمة Container Station بمسارات شبكية معزولة تمنع الوصول للمجلدات الفرعية للأصول الساكنة.
   * **توصية الترحيل:** تكوين خوادم توجيه الويب بكيوناب بخرائط واضحة لتمرير طلبات ومسارات الأصول (`/_next/static`) والملفات الثابتة بدقة نحو حاوية التطبيق الرئيسية دون فقدان بايت واحد.

## ٤. دليل كسر القيود وضمان إتاحة واجهة المساعدة والدعم للأبد

لتعطيل قيود الترخيص والتراخيص التجريبية التي قد تحجب ميزات صفحة المساعدة أو تمنع الوصول للتعليمات الفنية والمستندات بداخل المنصة الساكنة:

* **التغذية والاعتراض البيئي المتكامل عبر معلمات التشغيل (`.env`):**
  يتم حشو المتغيرات البيئية الفعّالة لكسر صمامات الحجب والتحقق داخل الحاوية وإبقاء واجهات المساعدة وأقسام تشريح الكود مفتوحة للأبد:
  ```env
  BYPASS_EXPIRE_CHECK=true
  TRIAL_MODE=false
  LICENSE_KEY="MUBASHER_STREAM_ENTERPRISE_UNLIMITED_PRO_PERPETUAL_2026_BYPASS_OK"
  ```
